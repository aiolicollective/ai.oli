Data Deletion Policy — /ai.oli collective

Last updated: October 2025

1. Overview

/ai.oli is a hybrid and creative collective using automated workflows (“n8n-social-aioli”) to publish and manage visual content on behalf of its members and collaborators. We act as a creative and automation service provider for multiple client accounts. No personal data is stored or reused.

2. Temporary data processing

All data handled by /ai.oli workflows — such as captions, media URLs, comments, or publication IDs — is processed only in memory, for the duration of the automation execution. Once the task is completed, no information is saved, archived, or accessible by any third party.

3. Revoking access

Since no user data is stored, deletion requests are generally unnecessary. However, if you have granted access to one of our applications (e.g., “n8n-social-aioli”) and wish to revoke it, please follow the instructions below:

  • Facebook / Instagram: Go to Settings → Apps and Websites → Find “n8n-social-aioli” → Remove Access.
  • LinkedIn: Settings → Data Privacy → Permitted Services → Remove “/ai.oli”.
  • Pinterest or other integrations: Revoke access directly from your account’s security or connected apps section.

4. Client-specific workflows

Each client automation is isolated within its own environment and API credentials. If a client terminates collaboration with /ai.oli, all related workflows, access tokens, and media are deleted immediately from our private infrastructure and associated cloud services (e.g., Cloudinary).
